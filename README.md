# Porsche 911 Turbo (1975) — 3D Interactive Showcase

A premium, interactive 3D web experience showcasing the legendary **1975 Porsche 911 Turbo (930)**. Built with Three.js, this project features high-fidelity modeling, cinematic camera animations, and real-time customization.

## ✨ Features

- **High-Fidelity 3D Model**: Accurate representation of the classic 1975 Porsche 911 Turbo.
- **Cinematic Experience**: Smooth intro animations and scroll-driven camera paths for an immersive walkthrough.
- **Real-Time Paint Customization**: Interactive color picker to change the car's paint finish on the fly.
- **Modern UI/UX**: Premium aesthetic with custom cursors, film grain overlays, and responsive design.
- **Performance Optimized**: Uses DRACO compression for fast loading of high-detail 3D assets.

## 🛠️ Tech Stack

- **Core**: [Three.js](https://threejs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Assets**: GLTF (with DRACO compression), HDRI Environment Maps via Poly Haven.
- **Styling**: Vanilla CSS with a focus on premium aesthetics and responsive layouts.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (latest LTS recommended)
- npm or yarn

### Installation

1.  **Clone the repository**:

    ```bash
    git clone <repository-url>
    cd threejs
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    ```

3.  **Run the development server**:

    ```bash
    npm run dev
    ```

4.  **Open the app**:
    Navigate to `http://localhost:5173` in your browser.

## 📦 Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the project for production.
- `npm run preview`: Previews the production build locally.
